# Prescription Scanner
## AI-Powered Medical Handwritten Prescription Recognition System

## 📌 Overview

Prescription Scanner is an AI-based web application that converts handwritten medical prescriptions into digital text using Optical Character Recognition (OCR) and Natural Language Processing (NLP). The system extracts medicine names, dosage details, frequency, and instructions from handwritten prescriptions.

The main objective of this project is to reduce errors caused by unclear handwriting, save time in prescription processing, and provide structured digital medical information.

---

# 🚀 Features

- Upload handwritten prescription images
- Image preprocessing for better accuracy
- OCR-based handwritten text extraction
- Medicine name identification
- Dosage and instruction extraction
- NLP-based text analysis
- Fuzzy matching for medicine correction
- Prescription history management
- PDF report generation
- User authentication system

# ⚙️ Working Process

1. User uploads prescription image
2. Image preprocessing improves image quality
3. OCR extracts handwritten text
4. NLP analyzes extracted information
5. Medicine names and dosage are identified
6. Results are displayed in structured format
7. Prescription report can be generated
